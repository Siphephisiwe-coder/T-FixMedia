# T-Fix Media Website

A premium, responsive business website built for T-Fix Media.

## 🚀 Overview
This project is a modern, conversion-focused website designed to showcase services and attract clients. It features a clean layout, strong visual hierarchy, and mobile responsiveness.

## ✨ Features
- Responsive design (mobile-friendly)
- Modern UI/UX
- Sticky navigation bar
- Hero section with call-to-action
- Services section with structured layout
- Contact section
- WhatsApp quick contact integration

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
